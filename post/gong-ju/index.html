1. 翻墙：就凭搜索引擎可以用谷歌，值了

2. 浏览器：QQ浏览器 + 油猴

（1）Microsoft Edge

谷歌浏览器的套壳，有阅读功能、朗读功能、老IE模式，通过微软帐号同步。

  • 优点：功能较多，没有被墙
  • 缺点：同步烂，曾经加过天朝特供广告

（2）Google Chrome

各种浏览器基本都是套它的壳，但自家更新肯定是最快的，通过谷歌帐号同步。

  • 优点：干净，更新快、稳定
  • 缺点：被墙

（3）Mozilla Firefox

除了谷歌内核就是火狐内核了，一般都是作为备用，通过火狐帐号同步。

  • 优点：隐私保护好，没有被墙
  • 缺点：天朝特供版恶心人，装国际版都不行

（4）Tor Browser

火狐浏览器的套壳，一般用于特殊用途，没有帐号功能，不能同步。

  • 优点：隐私保护无敌，可以访问暗网
  • 缺点：卡，被墙

总结：谷歌，有特殊需求就装Tor
